# BrainFlow UI/UX Design Spec — Screen Inventory (v2.1)

## Global
- **Theme:** dark mode with neon accents (neon-blue, violet, green).
- **Presentation:** all mockups are framed in an **iPhone device shell** with an iOS status
  bar and an iOS-style bottom tab bar. No Android frame exists yet.
- **Bottom navigation:** Home · Exercise · Progress · Profile.

## Screen 1 — Onboarding ("Start Today")
- Headline: "Train your brain. A few minutes a day."
- Single primary CTA: **"Start Today"** → drops the user straight into the first exercise.
- No account creation required up front. Frictionless entry, as intended.

## Screen 2 — Dashboard (Home)
- Greeting: "Good morning, Jonas."
- **Daily goal** ring: "60% done."
- **Streak** chip: "5-day streak 🔥".
- **AI Daily Pick card** (top of screen, prominent): "Your pick for today — *Box Breathing*,
  recommended for you." One-tap start, no manual choosing.
- Four **category tiles**: **Focus**, **Creativity**, **Emotional IQ**, **Wellbeing**.
  Each tile shows a time label of **"2–5 min"** and a difficulty dot.

## Screen 3 — Exercise
- Minimalist layout to reduce distraction.
- Prominent countdown **timer starting at 08:00** (an 8-minute Focus session in this frame).
- **"The Science"** expandable panel explaining the neurological benefit (e.g. prefrontal-cortex activation).
- Controls: pause, skip, end.

## Screen 4 — Progress
- User titled **"Brain Athlete — Level 4."**
- **Weekly Activity**: bar chart of sessions per day.
- **Brain Skills**: radar/spider chart. Axes labelled **Focus, Creativity, Emotional Intelligence, Wellbeing.**
- **Achievements**: badge grid (streak milestones, category mastery).
- No numeric score is displayed anywhere on this screen.

## Screen 5 — Upgrade prompt (state)
- Full-screen "Go Premium" sheet.
- Annotation in the frame: **"Trigger: Day 3 after first use."**
- Price shown: **€3.99 / month** (launch promo) or €49.99 / year.

## Screen 6 — B2B HR Dashboard (web)
- Web layout for HR admins.
- **Employee table**: lists staff **by name** (e.g. "Anna M. — 4-day streak — 82% weekly goal")
  with per-person engagement and category breakdowns.
- Filters: department, "low-engagement employees."
- Aggregate charts above the table.
