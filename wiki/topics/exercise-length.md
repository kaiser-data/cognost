# Exercise length

How long a guided exercise runs — specified four different ways.

**Current: undefined across teams.**

## The conflict (contradiction H)
| Source | Length |
| --- | --- |
| PRD §Scope (2026-01-22) | **2–5 minutes** |
| Content spec (2026-03-18) | **3–10 minutes** (Focus/Emotional IQ toward 10; Wellbeing as low as 3) |
| GTM (2026-04-25) | "**5 minutes**", every exercise |
| Design (2026-05-10) | Dashboard tiles say "**2–5 min**" but the Exercise screen timer starts at **08:00** (8 min) — design contradicts *itself* |

## Open issues
- The design's internal contradiction (tiles vs timer) sides with the content spec's longer range
  over the PRD. **Needs one agreed range**, then tiles, timer, GTM copy, and PRD aligned to it.
