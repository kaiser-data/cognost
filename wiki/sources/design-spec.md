# UI/UX Design Spec — Screen Inventory v2.1

**Source:** `raw/design-spec.md` · **Type:** DesignSpec · **Author:** Mara Köhler (Design/UX) · **Date:** 2026-05-10

Text transcription of the high-fidelity mockups, for checking design against PRD/roadmap/decisions. Reinforces several misalignments and contains an internal one.

## Claims asserted
- Ships a prominent **AI Daily Pick card** (violates PRD scope).
- Upgrade screen annotated **'Day 3'** (stale).
- HR dashboard lists employees **by name** (contradicts PRD).
- All frames are an **iPhone shell**, no Android (contradicts ADR-0001).
- Price shown **€3.99/mo**.
- Tiles say **'2–5 min'** but Exercise timer starts at **08:00** (internal contradiction).
- Progress screen has **no numeric Brain Score** (agreed feature missing).
- Category label **'Emotional IQ'** (tile) vs **'Emotional Intelligence'** (radar).
