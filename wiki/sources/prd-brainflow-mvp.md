# BrainFlow MVP PRD

**Source:** `raw/prd-brainflow-mvp.md` · **Type:** PRD · **Author:** Lena Hoffmann (Product) · **Date:** 2026-01-22

Canonical MVP spec. Defines vision, the Android-using primary persona (Jonas Weber), MVP scope, MoSCoW priorities, monetisation timing, B2B reporting rule, and the success metric.

## Claims asserted
- Exercises last **2–5 min** (§Scope).
- AI-powered Daily Picks + monetisation are **Won't have** for the MVP (§MoSCoW).
- Premium upgrade prompt on **Day 7** after first use (§Monetisation).
- B2B HR dashboard shows **anonymised, aggregated** data only (§B2B).
- Primary KPI: **30-day retention ≥ 40%** (§Success).
- Categories: Focus, Creativity, Emotional IQ, Wellbeing.
